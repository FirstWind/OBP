# Firebird Server (LAN)

## Режим работы
- Используется только Firebird Server 3.x (embedded не применяется).
- Подключение осуществляется к серверу в сети, локальный кэш не используется.

## Подключение клиента
Параметры подключения берутся из `config/app.ini`.
Строка подключения на клиенте задаётся как:

```
<host>:<path>\obp.fdb
```

По умолчанию используется порт 3050. В статус-баре показывается:

```
LAN: <host>:<port> / <db>
```

## Резервное копирование
Для резервного копирования применяется `gbak` на стороне сервера/админа. Пример:

```
gbak -user sysdba -pas <пароль> -b <host>:<path>\database.fdb backup.fbk
```

В v1 приложение не выполняет бэкап автоматически, но может вызывать внешние утилиты.
