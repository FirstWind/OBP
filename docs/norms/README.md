# Norms Pack (структура v1)

Стартовый пакет должен находиться в `docs/norms/<norms_id>/` и содержать:

- `manifest.json`
- `appendix10.json`
- `appendix11.json`
- `appendix12.json`
- `appendix13.json`
- `schemas/` с JSON Schema для каждого файла

Пакет берётся из `315 полный.pdf` и не “зашивается” в код.
